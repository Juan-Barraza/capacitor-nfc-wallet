# capacitor-nfc-wallet

NFC HCE payment plugin for MyDigitalWallet

## Install

To use npm

```bash
npm install capacitor-nfc-wallet
````

To use yarn

```bash
yarn add capacitor-nfc-wallet
```

Sync native files

```bash
npx cap sync
```
